# 🏀 GeneratingLeBron

A Python project that scrapes LeBron James quotes from the web and generates a **"LeBron Quote of the Day"** using simple natural language generation techniques.

This project was created as a way to learn the fundamentals of:

- Web scraping

- Data cleaning and preprocessing

- Git/GitHub workflows

- Basic natural language processing (NLP)

- Working with real-world text data

While the project started as a fun idea inspired by one of the greatest basketball players of all time, its real purpose was to gain hands-on experience building a complete data pipeline—from collecting data to generating new text.
